# 📐 피타고라스의 정리 (Pythagorean Theorem)

## ✅ 정의
직각삼각형에서,  
**두 직각 변의 제곱의 합은 빗변의 제곱과 같다.**

## 🧮 수식
만약 세 변의 길이가  
- a: 첫 번째 직각 변  
- b: 두 번째 직각 변  
- c: 빗변 (가장 긴 변)  
이라면,

a² + b² = c²

## 🎯 조건
- **직각삼각형**에서만 성립
- 항상 **가장 큰 변을 빗변(c)**으로 두고 계산해야 함

## 📝 예시
- (3, 4, 5) → 3² + 4² = 9 + 16 = 25 = 5² → ✅
- (6, 8, 10) → 6² + 8² = 36 + 64 = 100 = 10² → ✅
