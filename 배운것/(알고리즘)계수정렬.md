# 🧠 계수 정렬 (Counting Sort) – 아주 쉽게 배우기

## 📌 계수 정렬이란?

> 숫자끼리 비교하지 않고, **각 숫자가 몇 번 나왔는지만 세서 정렬**하는 알고리즘

---

## 📋 예시로 이해해보자!

입력값: `3 5 3 2 1`

1️⃣ 숫자가 몇 번 나왔는지 세기  
1 → 1번
2 → 1번
3 → 2번
5 → 1번

2️⃣ 숫자를 작은 순서대로 등장한 만큼 출력  
→ 1 2 3 3 5

---

## ⚙️ 어떻게 작동할까?

- **숫자 값 = 배열 인덱스**로 생각
- `counts[숫자]++` 형태로 빈도 저장
- 예: `counts[3] = 2` → 3이 2번 나왔다는 뜻

---

## ✅ 장점

- **매우 빠름** (O(N + K)) → N: 데이터 수, K: 숫자 범위
- 정수 데이터만 있다면 효율적

---

## ⚠️ 단점

- 숫자 **범위가 너무 크면 비효율적**
  - 예: 1 ~ 10억 → 메모리 낭비
- **음수는 다루기 힘듦**
  - 인덱스로 음수 사용이 불가능하므로 추가 조정 필요
